Projet d'Architecture et Développement Web

L'état actuel du projet contient les pages web (front) suivantes :

° home :
    La page "home" contient :
        - les nouveautés
        - les meilleures ventes
        - une suggestion d'achat.

    Un clic sur le slide / la jaquette Spider-Man redirige vers la page
    produit du jeu, un clic sur les autres éléments similaires redirigent
    vers une page "erreur 404".

° catalogue :
    La page "catalogue" contient :
        - un filtre permettant de lister les jeux selon
          un ou plusieurs paramètres (non fonctionnel)
        - un listing des produits.

      Un clic sur la jaquette Spider-Man redirige vers la page
      produit du jeu, un clic sur les autres éléments similaires redirigent
      vers une page "erreur 404".

° log :
    La page "log" contient un formulaire de connexion (non fonctionnel).

    Les informations demandées pour la connexion doivent être bien renseignées,
    condition sine qua non à l'apparition du bouton d'envoi du formulaire.

° sign :
    La page "sign" contient un formulaire d'inscription (non fonctionnel).

    Les informations demandées pour l'inscription doivent être bien renseignées,
    condition sine qua non à l'apparition du bouton d'envoi du formulaire.

° game :
    La page "game" contient diverses informations sur un jeu en particulier (développeur, trailer, ...).

° user :
    La page "user" contient :
        - les informations d'utilisateur
        - un lien "Modifier mon compte" qui redirige vers une page de modification des informations du compte.

    Un clic sur le lien "Modifier mon compte" redirige vers une page ("account") qui
    permet de modifier les informations d'un compte.
    Si une ou plusieurs informations sont modifiées, il suffit que l'une d'elle soit
    mal renseignée pour empêcher l'apparition des boutons d'annulation et d'envoi.

° account :
    voir ci-dessus.

° cart :
    La page "cart" contient :
        - la liste des jeux présents dans le panier
        - un bouton pour augmenter / diminuer le nombre d'exemplaires
          d'un jeu (>=1)
        - un bouton pour supprimer le produit (non fonctionnel)
        - un bouton pour payer le produit (non fonctionnel).

    Un clic sur la jaquette Spider-Man redirige vers la page produit du jeu.
